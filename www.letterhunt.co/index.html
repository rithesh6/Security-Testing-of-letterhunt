Frequently Asked Questions

Got a question? We've got answers.

Do I need programming skills to use this?

No. All data is made available through Airtable and thus it's easy to create custom filters without coding skills.

Is the data always up-to-date?

No, all data is cached. For most use-cases it won't make a big difference.

How often is the data updated?

Currently, all data sets are updated once a month. This should be sufficient for most use-cases.

Can i export the database in $99 plan?

Exporting data to CSV is not possible in the $99 plan. That feature is available with a $250 planer.

Can i get a Refund?

Refund is not allowed.
